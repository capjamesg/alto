Artemis Auth

Artemis Auth is an IndieAuth server.

Using Artemis Auth, you can verify your identity using your domain name, and issue tokens for services to use that support IndieAuth.

To get started, add the following meta tags to your website:

<link rel="authorization_endpoint" href="{{ AUTH_SERVER_URL }}">
<link rel="token_endpoint" href="{{ TOKEN_SERVER_URL }}">

When you sign into a service that supports web sign in (i.e. the Artemis web reader), you will be able to log in using Artemis Auth.
